任务目标
============

梅林的另一批旧魔法书刚被运到他在图书馆的办公室中
书中有一些提及了gsh(谷胱甘肽 Glutathione)
请找出它们，并且把书的名字输出到一个列表中
注意，它可能大写，可能小写，可能大小写都有

有一些书涉及高阶机密，你没有权限阅读它们，请忽略掉它们

备注
------

只需要输出书名字的清单，不要涉及其中内容


秘籍
===============

grep STRING FILE1 ... FILEn
  在文件FILE1-FILEn当中搜索字符串STRING，并且输出相关行
  这个命令有一些有趣的选项:
    -l : 不输出相关行，而是输出带有字符串STRING的文件名
    -i : 忽略大小写

COMMAND  >  FILE
  把命令COMMAND的标准输出(stdout)输出到文件FILE中，文件将被覆写

COMMAND  2>  FILE
  把命令COMMAND的标准错误(sttderr)输出到文件FILE中，文件将被覆写

/dev/null
  一个通常情况下永远为空的虚拟文件，所有被输出到这里的东西会消失


扩展阅读
------------

所有命令都有以下两种手段往屏幕输出信息
  - 标准打印 ("stdout")
  - 错误打印 ("stderr")
默认情况下，两种方法输出的信息 都会被显示到屏幕上
(stdout与stderr是有区别的！感兴趣可以自行查阅资料)

然而你可以去单独重定向它们到文件
