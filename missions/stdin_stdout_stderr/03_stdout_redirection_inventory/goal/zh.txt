任务目标
============

Merlin有一些旧魔法书放在了他图书馆的办公室里
你需要把一份包含所有《魔法书》的目录
保存在一个文件名为"inventory.txt"的文件中
【把这份文件放到抽屉里！】
(抽屉=drawer 办公室=office)

秘籍
===============

ls FILE1 ... FILEn
 显示作为参数给出的文件的列表
 当你使用通配符时，会很爽~

COMMAND  >  FILE
  把命令COMMAND的输出放到文件FILE中，而不是打印在屏幕上

less FILE
  展示文件的内容，一次一页

  重要的快捷键:
    q         quit 退出
    Space     往下翻页
    / STRING  查找字符串 STRING
    n         前往下一个匹配STRING的位置
