任务目标
============

使用"|"连接起来的命令在迷宫中找到被藏起来的钻石，之后把它放到你的箱子中
迷宫 = Maze

提示
------

然而文件名当中并没有 "钻石"~ 想必钻石藏在文件里面吧


秘籍
===============


find
  在当前目录找文件file/目录directory

  你可以附加一些关于文件名字、大小、修改日期的的条件

grep [OPTIONS] STRING FILE1 ... FILEn
  在文件当中搜索字符串

  常用选项[OPTIONS]:
    -l 仅仅打印出来文件名

xargs COMMAND
  从stdin当中读取行，并且运行COMMAND
  ``xargs`` 与 ``|`` 一起使用，妙妙妙


备注
================  
xargs 是一个用于传递参数的过滤器，由于很多命令不支持直接使用|传参
所以 xargs 是在连接命令时一个非常重要的命令

除此之外，xargs还可以用来转换输出格式等等
xargs 的功能很多，推荐去查阅其手册学习使用方法

在本次任务当中，你可能需要使用xargs的以下格式
  xargs -i COMMAND {} otherargs
  指的是把{}用xargs捕获到的输出替换
