任务目标
============

在花园的迷宫中找到红宝石，并且把它放到你的箱子里
花园 = Garden 迷宫 = Maze 红宝石 = Ruby

提示
----

然而文件名当中并没有 "ruby"
幸运地是迷宫当中没有多少文件，你可以一个个地检查那些 **file**(文件)
(不是*directories*目录哦), 手动找出到底哪个文件包含字符串"ruby"


秘籍
===============

find
  在当前目录找文件file/目录directory

  你可以附加一些关于文件名字、大小、修改日期的的条件
  以下这个条件有点意思~
    -type TYPE (文件类型)

cat FILE
  把文件内容输出到屏幕

man COMMAND
  展示COMMAND的命令使用手册

  重要的快捷键:
    q           退出
    Space       滚动页面
    / STRING    搜索STRING
    n           找到下一个匹配项
              

备注
------

上面有关`find`命令的描述是故意没说全的，你可以在手册当中找到更多信息
(你可以在里面看到更多关于-type的信息哦！)
