任务目标
============

在你的箱子当中创建文件"journal.txt"并且写点东西
你可以使用这个文件记些笔记，后续的任务没准也可以用它帮助完成~

详述
-------

``nano`` 是一个命令行文本编辑器。
无论何时，只要当你需要在shell当中编辑文件的时候就可以使用~


秘籍
===============

nano FILE
  在shell当中编辑文档FILE
  (当文件不存在的时候，会自动创建)

  快捷键会被显示在屏幕下方 (符号 "^" 
  是 "Control"的意思(Ctrl键)).
  下面是一些非常重要的快捷键(ctrl与后面的字母一起按):
    Control-x     quit 退出
    Control-o     save 保存
    Control-w     search for a string 查找字符串

  注意: 不要用Control-s Control-z!
