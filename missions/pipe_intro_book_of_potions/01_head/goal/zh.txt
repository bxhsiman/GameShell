任务目标
============

一位名叫Servillus的健忘的老隐士在一个山洞里安营扎寨
他有一本旧的皮装药剂书
去去山洞帮他复习一下他著名的凉茶配方
(山=Mountain 山洞=cave 药剂书=Book_of_potions)

为了完成任务,你需要在山洞中，陪在Servillus旁边
**并且** 你在执行 ``gsh check`` 命令前的最后一个指令必须展示凉茶配方
(包括它的标题，但是不能带有其他东西)

提示:你不可以改变药剂书的内容


秘籍
===============

cat FILE
  展示FILE文件的内容

head FILE
  打印FILE文件的前10行

head -n K FILE
  打印FILE文件的前k行

提示
------

如果要处理的文件不在当前目录，"FILE"则需要带上其目录的名字
